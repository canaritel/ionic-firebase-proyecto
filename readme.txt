Proyecto final de Antonio González Bonilla para PGL
Se envía sin la caperta "node" para reducir el tamaño del Proyecto

Para poder ejecutarlo la primera vez (sin la carpeta node) realizo los siguientes pasos:
1. npm install
2. ng update
3. npm update
5. ionic serve

La aplicación integra la cámara, también el mapa y el sistema de base de datos con Firebase, así como otras funcionalidades que hemos realizado en clase.
